# BrowserPilot MCP Server

BrowserPilot MCP Server is an open source Model Context Protocol (MCP) server for browser automation research and local testing.

It exposes tools for page navigation, screenshot capture, DOM inspection, and controlled browser automation. The project is published on GitHub under the MIT License.

The README documents setup instructions, MCP configuration, and a rate limit recommendation of 30 requests per minute for polite use. This fixture does not include a pricing page or privacy policy.
