# Anti-Clone Strategy

Surface cloning is possible because AOI is public and published to PyPI. The goal is not to pretend otherwise.

Defensible moat strategy:

- source trace quality and coverage;
- freshness and field completeness over time;
- ExecutionReceipt memory;
- local Probe-before-Use history;
- GitHub issue feedback loop;
- vertical-specific calibration;
- private ranking kernels and anti-gaming rules;
- conservative public claims that build trust;
- distribution surfaces such as PyPI, GitHub, Hugging Face, and eventually MCP Registry.

Public AOI may expose the router shape and public schemas. Private operational calibration should remain non-public.
