# Community Manual Post Queue

Package 8P creates a manual copy-paste queue for conservative feedback posts.

The queue is written to:

- `public_launch/wave2/COMMUNITY_MANUAL_POST_QUEUE.md`
- `public_launch/wave2/COMMUNITY_MANUAL_POST_CHECKLIST.md`

No automatic post is performed.

Post one venue at a time and keep the wording modest:

- read-only
- source-traced
- registry metadata candidates
- not verified
- not security certified
- not a quality guarantee
- no purchasing advice
