# Conservative Community Feedback Launch

The public beta launch copy is intentionally modest.

Allowed framing:

- read-only MCP/API benchmark
- objective-based ranking
- Official MCP Registry metadata candidates
- source traces, missing fields, score components, and decision receipts
- please test with golden queries and open issues

Forbidden framing:

- verified MCP servers
- safe MCP servers
- security certified
- quality guaranteed
- official standard
- production-ready
- purchasing advice

Package 8O creates copy-paste drafts for manual platforms. It does not auto-post to HN, Reddit, Product Hunt, OpenAI Developer Community, or MCP community sites.
