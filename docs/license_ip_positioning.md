# License and IP Positioning

AOI currently uses the MIT License. MIT is permissive, which can support adoption and integration, but it can also allow copying and forking under the license terms.

Non-legal positioning options:

- keep open source for adoption;
- dual-license later;
- keep the public router open and commercial/private kernel closed;
- consult a legal professional for patents, trademarks, copyright, and commercial licensing.

This document is not legal advice. Treat it as an operational reminder before increasing distribution through MCP Registry or broader community posting.
