# Dashboard Refresh Claim Boundaries

ROE-21 dashboard refresh artifacts may say:

- local/static dashboard refresh
- feedback bridge summary
- AgentSec executed locally
- QIRA/DataCapsule/Portfolio skipped with missing artifact context
- ALLOW/HOLD/BLOCK counts
- stale external share pack notice
- known limits

ROE-21 dashboard refresh artifacts must not claim:

- security certification
- code correctness proof
- legal, privacy, license, or eval-clean proof
- quality guarantee
- product readiness
- external action authorization
- skipped candidates succeeded
- live execution or external pilot completion
