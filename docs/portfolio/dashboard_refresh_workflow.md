# Dashboard Refresh Workflow

1. Read the ROE-20 feedback second-run receipt.
2. Read the prior ROE-16 dashboard artifacts.
3. Create a refresh delta.
4. Create four status cards: AgentSec, QIRA, DataCapsule, and Portfolio.
5. Create a timeline update and feedback memory summary.
6. Create a stale notice for the ROE-17 external share pack.
7. Run redaction and claim-boundary checks.
8. Run the ROE-21 gate.

The workflow preserves skipped and HOLD states. A skipped candidate is not converted into success.
