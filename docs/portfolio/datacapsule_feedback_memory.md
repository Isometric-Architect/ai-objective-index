# DataCapsule Feedback Memory

DataCapsule feedback memory records what a reviewer, owner, or data steward should revisit after a pilot receipt.

Feedback can suggest:

- manifest schema updates
- new public-safe fixtures
- new generic negative controls
- license/terms evidence requests
- privacy/consent evidence requests
- evaluation split-policy evidence requests
- claim-boundary changes

Feedback memory does not post externally, fetch URLs, upload data, train models, call APIs, or authorize actions.
