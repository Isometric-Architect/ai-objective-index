# External Share V2 Claim Boundaries

The V2 share pack may say:

- static local demo
- refreshed from ROE-21
- AgentSec executed/incorporated locally
- QIRA/DataCapsule/Portfolio skipped_missing_artifact
- `external_action_count = 0`
- known limits
- claim boundary

The V2 share pack must not claim:

- external pilot completion
- product launch
- security certification
- code correctness proof
- legal, privacy, license, or eval-clean proof
- quality guarantee
- product readiness
- external action authorization
- skipped candidates succeeded
- live connector readiness

Skipped candidates are HOLD, not failure and not success.
