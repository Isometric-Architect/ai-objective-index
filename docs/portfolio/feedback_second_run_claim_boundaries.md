# Feedback Second-Run Claim Boundaries

ROE-20 may produce local receipt updates and skipped-candidate reports.

It must not be described as:

- an external pilot
- security certification
- code correctness proof
- legal, privacy, license, or eval-clean proof
- a quality guarantee
- product readiness
- no merge, deploy, publish, upload, training, or external action authorization

Feedback can improve explanations, add fixture candidates, add negative-control candidates, or request evidence. Feedback cannot certify safety, prove correctness, clear legal/privacy/license risk, or authorize external action.
