# No-Contact Feedback Launch Policy

If there are no existing reviewers, ROE-18 still remains manual-first.

Allowed:

- prepare drafts;
- choose likely reviewer personas;
- review claim boundaries;
- decide whether to contact someone later;
- keep all replies as local feedback packets.

Not allowed in this package:

- automatic email;
- automatic DM;
- community auto-post;
- GitHub issue creation;
- scraping reviewer lists;
- external API calls;
- upload or deployment;
- live form service.

When no reviewer is available, keep the package as an internal planning artifact and move to feedback-reply intake only after a human-provided response exists.
