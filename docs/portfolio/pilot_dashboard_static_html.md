# Pilot Dashboard Static HTML

`pilot_dashboard/RESIDUALOPS_PILOT_DASHBOARD.html` is a static local artifact.

It contains inline CSS only. It does not include external JavaScript, external CSS, CDN links, network calls, or forms. It is meant for local inspection of generated receipts and gate status.

The page includes a visible claim-ceiling banner and tables for:

- vertical status cards;
- gate decisions;
- feedback memory.

The HTML does not contain raw private data, credentials, tokens, or private kernel values.
