# Pilot Dry-Run Claim Boundaries

The dry-run may claim:

- local/sample dry-run
- local/offline receipt generation
- vertical route result
- ALLOW/HOLD/BLOCK count
- redaction status
- feedback memory status

It is not:

- not an external pilot
- not security certification
- not code correctness proof
- not legal opinion
- not privacy audit
- not license clearance
- not evaluation-cleanliness proof
- not quality guarantee
- not product-readiness claim
- no external action authorization
