# Pilot Feedback Intake Form

The feedback form template asks reviewers what they reviewed, which vertical they inspected, what was confusing, which claim felt too strong, which ALLOW/HOLD/BLOCK finding looked wrong, and what evidence they would want.

It also asks whether the reviewer consents to a future local/offline pilot using a redacted artifact.

The form must not collect tokens, passwords, private keys, raw PII, private datasets, private kernel values, credentials, or live connector secrets.

The form is static text. ROE-18 does not create a live form service and does not submit responses anywhere.
