# Pilot Outreach Claim Boundaries

ROE-18 is a feedback-request package only.

It is not:

- security certification;
- code correctness proof;
- legal, privacy, license, or eval-clean proof;
- product readiness review;
- quality guarantee;
- adoption request;
- external action authorization;
- automated outreach.

Allowed framing:

- static demo;
- local artifact;
- feedback request;
- claim boundary;
- ALLOW/HOLD/BLOCK;
- known limits;
- no external action.
