# Pilot Outreach Operator Checklist

Before using any draft:

1. Read `pilot_outreach/PILOT_OUTREACH_CLAIM_BOUNDARY.md`.
2. Read `pilot_outreach/PILOT_OUTREACH_DO_NOT_SEND_GUARD.md`.
3. Choose one persona.
4. Copy one draft manually.
5. Replace TODO links only after review.
6. Do not paste tokens, credentials, passwords, private keys, raw PII, private datasets, or private kernels.
7. Do not promise certification, proof, product readiness, quality, adoption, or external action.
8. Ask for feedback, not adoption.
9. Record any reply as a local feedback packet.
10. Do not auto-send.
