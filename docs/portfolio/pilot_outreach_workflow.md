# Pilot Outreach Workflow

ROE-18 keeps outreach manual.

1. Generate the outreach pack.
2. Read the claim boundary.
3. Choose one reviewer persona.
4. Copy one draft manually.
5. Replace TODO link entries only after review.
6. Ask for feedback on the static demo/share pack.
7. Record any reply as a local feedback packet.
8. Route feedback through the triage taxonomy.
9. Plan a local/offline second-run only when consent and redaction are clear.

This workflow performs no automatic sending, posting, issue creation, upload, deployment, live connector call, or external action.
