# Pilot Run Checklist

Use this checklist before a future owner-consented pilot:

1. Receive a local file or pasted metadata.
2. Confirm owner consent and scope.
3. Confirm no credentials, `.env` content, raw secrets, or private kernel values are included.
4. Run redaction preflight.
5. Route the artifact to AgentSec, QIRA, DataCapsule, or manual triage.
6. Create the local run plan.
7. Run only the selected local/offline pilot packager.
8. Produce a receipt, reviewer readout, and feedback memory entry.

Do not post, merge, deploy, upload, train, fetch, clone, call GitHub APIs, call external APIs, run live MCP/tool calls, execute external tools, or use credentials.
