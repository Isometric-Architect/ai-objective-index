# ResidualOps Feedback Memory Index

The feedback memory index unifies pilot follow-up signals across AgentSec, QIRA-Code ReleaseGate, and DataCapsule.

It records whether each vertical suggests:

- adding a negative control
- adding a public-safe fixture
- updating a policy, behavior contract, or manifest schema
- requesting further evidence
- changing claim boundaries

The index does not post issues, create pull requests, contact owners, call GitHub APIs, or mutate external repositories. It is a local planning artifact.
