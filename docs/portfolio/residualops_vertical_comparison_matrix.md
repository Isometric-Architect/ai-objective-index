# ResidualOps Vertical Comparison Matrix

The vertical comparison matrix places AgentSec, QIRA-Code ReleaseGate, and DataCapsule side by side.

Columns include:

- vertical
- reviewed object
- input type
- check type
- external action used
- live execution used
- ALLOW/HOLD/BLOCK counts
- primary hold or block reason
- feedback memory status
- claim boundary
- next action

The matrix is for local/offline reviewer orientation. It does not claim product readiness, quality, security, legal sufficiency, privacy sufficiency, license clearance, evaluation cleanliness, or external action authorization.
