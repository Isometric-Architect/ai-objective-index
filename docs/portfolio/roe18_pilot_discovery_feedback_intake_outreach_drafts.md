# ROE-18 Pilot Discovery / Feedback Intake Outreach Drafts

ROE-18 creates manual-only feedback discovery materials for ResidualOps.

It prepares:

- reviewer persona map;
- conservative copy-paste feedback request drafts;
- local feedback intake form template;
- feedback triage taxonomy;
- link pack;
- FAQ;
- claim boundary sheet;
- operator checklist;
- do-not-send guard.

ROE-18 does not send email, DM anyone, post to communities, create GitHub issues, call GitHub APIs, call external APIs, fetch URLs, scrape, upload, deploy, or create a live form service.

The drafts ask for feedback on static/local artifacts. They do not claim certification, proof, product readiness, quality guarantee, legal/privacy/license clearance, eval-clean status, or external action authorization.

Run:

```powershell
python -m ai_objective_index.portfolio.pilot_outreach_drafts --generate
python -m ai_objective_index.portfolio.roe18_outreach_gate
```
