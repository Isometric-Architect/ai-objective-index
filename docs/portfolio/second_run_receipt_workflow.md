# Second-Run Receipt Workflow

ROE-15 follows this local workflow:

1. Load ROE-14 feedback packets, classifications, and second-run plans.
2. Load local AgentSec, QIRA, and DataCapsule pilot receipt samples.
3. Generate local second-run vertical results.
4. Generate before/after deltas.
5. Update feedback memory.
6. Generate a reviewer readout and artifact index.
7. Run redaction and the ROE-15 gate.

The workflow updates explanations, fixture candidates, evidence requests, and next actions. It does not upgrade decisions to certification or action authorization.
