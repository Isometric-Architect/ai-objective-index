# Private Deployment Review

Review the private deployment before any public switch.

1. Open the private GitHub repo.
2. Open the private Hugging Face Space.
3. Test query: `browser automation MCP server`.
4. Open the private Hugging Face Dataset.
5. Confirm all pages say the registry records are not verified and not security certified.
6. Confirm no token or credential is visible.
7. Confirm there is no claim of public release.
8. Decide whether to keep private, share with trusted reviewers, or prepare a later manual public switch.

Private deployment is a staging state. It is not a public release and not a product-readiness guarantee.
