# Public Launch Policy

AOI can move from private staging to public only through explicit human choice.

Rules:

- Dry-run is the default.
- Execute requires `AOI_PUBLIC_LAUNCH_CONFIRM=YES`.
- Public visibility does not create a GitHub Release.
- Public visibility does not post community announcements.
- Public visibility does not submit AOI to MCP Registry.
- Public visibility does not claim verified MCP servers, safe MCP servers, security certification, quality guarantee, purchasing advice, or official standard status.

`public_beta_mcp` remains a source-traced registry metadata candidate dataset. It is useful for testing, comparison, and feedback, but it is not verification or certification.
