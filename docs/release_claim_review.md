# Release Claim Review

Package 7B adds a simple claim audit for public-facing Markdown.

## Allowed Claims

- AOI is a read-only MCP/API benchmark tool.
- AOI is an objective ranking and comparison engine.
- AOI surfaces source traces and missing fields.
- AOI supports local `sample`, `generated`, and `integrated` data scopes.
- AOI is experimental public beta software.

## Forbidden Claims

- AOI is an official standard.
- All AI will use AOI.
- AOI guarantees quality.
- AOI provides legal, security, compliance, financial, medical, purchasing, procurement, or professional certification.
- AOI automatically buys, books, logs in, sends email, submits forms, purchases, connects accounts, verifies suppliers, or signs contracts.

## Productization Mode

Productization Mode allows implementation and release preparation. Public product, security, legal, market, or readiness claims require separate product evidence.

Run:

```powershell
python -m ai_objective_index.release_claim_audit
```
