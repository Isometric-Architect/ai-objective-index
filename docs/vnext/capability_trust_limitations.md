# Capability Trust Limitations

CapabilityTrust is a conservative route-readiness estimate.

It is not:

- a quality guarantee
- security certification
- legal, financial, medical, or purchasing advice
- official registry ranking
- proof that a capability works
- permission to pay, book, log in, send email, purchase, submit forms, or sign contracts

Extracted and source-traced metadata can be stale or incomplete. Package 9B does not execute probes, run a gateway, scan live systems, crawl websites, or call external LLM APIs.
