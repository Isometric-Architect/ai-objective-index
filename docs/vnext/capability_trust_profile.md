# Capability Trust Profile

`CapabilityTrustProfile` names the public components used by the local demo profile:

- objective fit
- source trace coverage
- evidence quality
- integration readiness
- policy clarity
- documentation clarity
- freshness
- missing field penalty
- residual risk penalty
- unsupported claim penalty
- forbidden action penalty

Package 9B uses `demo_profile_v0_1` only. The demo profile is not objective truth, not an anti-gaming signal, and not a security signal.

Actual future production weights, thresholds, negative-control seeds, and anti-gaming rules should remain private.
