# MCP Registry Resume After vNext

MCP Registry submission remains gated after Package 9F.

The registry path can resume only after:

- the package version decision is made;
- a local wheel/sdist builds cleanly;
- the package is uploaded and verified on PyPI in a later package;
- `.mcp/server.json` matches the package name and version;
- registry tooling and authentication pass;
- explicit submission confirmation is present.

AOI must continue to describe MCP Registry data as source-traced metadata candidates, not verified servers, safe tools, security certification, or quality guarantee.
