# Migration From v0.2 To v0.3

v0.2 remains the public beta ranking engine.

v0.3 should add vNext routing concepts incrementally:

1. ObjectiveCard as richer routing input.
2. CapabilityCard as richer candidate metadata.
3. ResidualCredit as safer post-evaluation accounting.
4. ProbeCard as pre-use check metadata.
5. ExecutionReceipt as feedback loop record.
6. CapabilityGraph as relationship layer.

The package name remains `ai-objective-index`.

PyPI upload and MCP Registry submission remain paused until alignment passes.
