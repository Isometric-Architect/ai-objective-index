# Objective Router Limitations

The Objective Router API is not:

- security certification
- a quality guarantee
- a verified registry
- product readiness certification
- purchasing advice
- legal, financial, or medical advice
- an automatic security gateway

Metadata can be stale or incomplete. Source traces can be partial. Package 9C does not run probes, execute tools, crawl websites, fetch live URLs, call external LLM APIs, upload to PyPI, submit to MCP Registry, or post to communities.
