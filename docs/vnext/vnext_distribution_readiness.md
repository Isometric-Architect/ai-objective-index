# vNext Distribution Readiness

The distribution surface includes:

- `README.md`
- `pyproject.toml`
- `.mcp/server.json`
- `api/openapi.json`
- `api/vnext/objective_router_openapi.json`
- `data/generated_mcp_tools_manifest.json`
- vNext docs
- public launch reports

These surfaces must agree that AOI is read-only, source-traced, objective-relative, and conservative. They must not claim verified capability, safety, security certification, quality guarantee, product readiness, purchasing advice, or gateway action authorization.

The current version decision is intentionally held until the owner chooses `0.3.0` or `0.3.0a1` for vNext distribution.
