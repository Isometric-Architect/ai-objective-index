```text
Find a cheap image generation API with commercial-use terms.
```

```text
Compare MCP browser automation servers by docs and source trace.
```

```text
Explain why the top-ranked OCR API scored highest.
```

```text
List missing fields for the top three AI coding assistant candidates.
```
