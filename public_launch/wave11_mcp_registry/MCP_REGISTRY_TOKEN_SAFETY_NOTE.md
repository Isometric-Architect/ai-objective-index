# MCP Registry Token Safety Note

`mcp-publisher` may use GitHub browser or device authentication.

- Do not paste GitHub tokens into ChatGPT/Codex chat.
- Do not commit credentials.
- Do not store secrets in this repository.
- Do not create or commit token helper files.
- If a token was exposed, revoke it immediately.

Registry publication is metadata distribution only. It is not verification, security certification, a quality guarantee, product-readiness evidence, purchasing advice, or action authorization.
