# GitHub Issue Labels Plan

Repository: https://github.com/Isometric-Architect/ai-objective-index

Dry-run is the default. Execute mode creates or updates labels with GitHub CLI if authenticated.

- `public-beta`: Public beta feedback and triage. color `#0E8A16`
- `failed-query`: A query produced weak, missing, or surprising results. color `#D93F0B`
- `wrong-field`: A field appears incorrect or unsupported. color `#B60205`
- `scoring-dispute`: Ranking or score component dispute. color `#FBCA04`
- `missing-source-trace`: A result needs better source trace coverage. color `#5319E7`
- `install-failure`: Local API, MCP, or demo setup issue. color `#C5DEF5`
- `docs-confusion`: Documentation is confusing or incomplete. color `#006B75`
- `add-tool`: Suggestion for a new tool/API/SaaS/MCP object. color `#1D76DB`
- `good-first-feedback`: Small, actionable public beta feedback item. color `#7057FF`

No labels are deleted. No tokens are printed or requested.
